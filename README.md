# Fixture Scraper

A simple Node.js application to scrape football fixtures from a given website.

## Requirements

- Node.js
- Typescript
- npm or yarn

## Setup

1. Clone this repository
2. Run `npm install` to install dependencies.

## Running the app

To start the scraper, run:

npm start

## Updating tha app

To change any code,run:

npm run build
npm start
